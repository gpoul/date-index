date-index creates a graph of the number of posts made over time using the Google Chart API.

== Installation ==

1. Upload `date-index.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<!-- date-index -->` in your pages where the graph should be displayed
